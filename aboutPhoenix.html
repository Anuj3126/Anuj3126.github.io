About Phoenix


This project is a music player web application with a sleek and modern design.
It has a responsive navigation bar with a logo, home and about links.
The main content area consists of a playlist with song items, each displaying a thumbnail, title, artist, and duration.
Users can play, pause, skip and rewind using the control buttons on the player.
There is also a progress bar that shows the current playback time and allows users to skip to a specific point in the song.
The bottom of the page contains information about the application, including the developer's social media links.
Overall, the project is a functional and aesthetically pleasing music player that provides an enjoyable listening experience for users.
